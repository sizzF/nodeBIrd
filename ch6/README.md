# vue simple sns project
- 인프런의 vue로 NodeBird SNS 강좌를 보면서 만들어본 토이 프로젝트입니다.
- 프론트엔드 : Vue, Nuxt, Vuex, Vuetify, Axios
- 백엔드 : Node, Express, Sequelize(MySQL), passport, sharp
- 배포 : AWS(EC2, S3, Route53)
- URL : https://nodebird.site/
